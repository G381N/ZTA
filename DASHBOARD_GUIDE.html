🛠️ Management Tools

Quick Start Script

Use the interactive menu system:

./quick_start.sh

Available Commands:

  • ./quick_start.sh start - Start system in foreground
  • ./quick_start.sh background - Start system in background
  • ./quick_start.sh stop - Stop the system
  • ./quick_start.sh status - Check system status
  • ./quick_start.sh dashboard - Open dashboard in browser
  • ./quick_start.sh test - Test all dashboard cards

Dashboard Manager

Comprehensive system management:

python3 dashboard_manager.py

Features:

  • ✅ Validate all APIs
  • 📊 System information and statistics
  • 🗑️ Delete learned history (reset training)
  • 🛡️ Trust score testing

🎯 Training Process

📚 Learning Phase (First 2 Days)
The system learns your normal behavior patterns. During this time, anomaly detection is disabled while it establishes your baseline.

What the System Learns:

  • 🕘 Your usual work hours (8 AM - 6 PM by default)
  • 💻 Applications you normally use
  • 🌐 Network connection patterns
  • ⏱️ Session duration habits
  • 🔄 Application switching patterns
🚀 Active Monitoring Phase
After training completes, the system activates real-time anomaly detection and dynamic trust scoring.

📱 Mobile Access

Access from Mobile Devices

1
Find your computer's IP address:
ip addr show | grep 'inet ' | grep -v '127.0.0.1'
2
Access from mobile browser:
http://YOUR_IP_ADDRESS:8080
Note: Make sure your mobile device is on the same network as your computer.
